# BTA Awards 2026

Premium awards website with Telegram integration.

Last updated: 2025-03-04

## Features

- ✅ 3 Languages (UZ/RU/EN)
- ✅ Responsive Design (Mobile/Tablet/Desktop)
- ✅ Premium Animations & Effects
- ✅ Telegram Bot Integration
- ✅ Online Registration
- ✅ Venue Information
- ✅ Gallery with Previous Years

## Live Demo

🌐 **Website:** https://bta-awards-2026.vercel.app

## Tech Stack

- **Frontend:** HTML5, Tailwind CSS, Vanilla JS
- **Build Tool:** Vite
- **Animations:** GSAP
- **Smooth Scroll:** Lenis
- **Deployment:** Vercel
- **Notifications:** Telegram Bot API
